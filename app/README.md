# תיק אתר אינטראקטיבי — מערכות מידע (גב-ים)

אפליקציית web מודרנית (2026) למילוי וניהול **תיק אתר מערכות מידע**, כתחליף אינטראקטיבי
למסמך ה-Word הסטטי. עברית RTL מלאה, מותג גב-ים, ושמירה מקומית בדפדפן.

## יכולות
- **מילוי טפסים** לכל 9 הפרקים + בקרת מסמך + פרטי אתר (זהה לתבנית ה-Word).
- **שמירה אוטומטית** בדפדפן (localStorage) — שורד רענון, ללא שרת.
- **ריבוי אתרים** — פרופיל נפרד לכל אתר (חדש / שכפל / שנה שם / מחק).
- **ייצוא / ייבוא JSON** — גיבוי ושיתוף בין מחשבים/אנשים.
- **ייצוא ל-Word (.docx)** — מפיק מחדש את המסמך הממותג מהנתונים.
- **הדפסה / שמירה כ-PDF** — פריסת A4 נקייה עם לוגו ושער.
- **חיפוש מהיר** (Ctrl/⌘+K) על פני פרקים, שדות ובקרות.
- **מחווני השלמה** כולל ולכל פרק · **מתג "הצג דוגמאות"** · **מצב כהה/בהיר**.
- ולידציות עדינות (IP / דוא"ל / תאריך) · נגישות · רספונסיבי.

## הרצה (פיתוח)
```powershell
cd app
npm install        # פעם ראשונה בלבד
npm run dev        # http://localhost:5173
```

## בנייה לפריסה (build סטטי)
```powershell
npm run build      # פלט ל-app/dist
npm run preview    # תצוגה מקדימה של ה-build
```
התיקייה `app/dist` היא אתר סטטי — ניתן לפתוח/לארח בכל שרת קבצים.

## בדיקות וטיפוסים
```powershell
npm run test       # Vitest (28 בדיקות)
npx tsc --noEmit   # בדיקת טיפוסים
```

## ארכיטקטורה (schema-driven)
| תיקייה | תפקיד |
|--------|-------|
| `src/schema/` | **תוכן התיק** כ-data (מקור האמת, מועתק 1:1 מ-`../generate.js`) |
| `src/engine/` | מנוע רינדור גנרי: שדה, טבלה חוזרת, צ'קליסט, kv, bullets, box, callout |
| `src/store/`  | state · שמירה ל-localStorage · ריבוי אתרים · ולידציה · השלמה · ייצוא/ייבוא |
| `src/ui/`     | shell, ניווט, header, בורר אתרים, theme, חיפוש |
| `src/print/`  | תצוגת הדפסה/PDF + ייצוא docx |

הוספת/שינוי שדה = עריכת קובץ הסכמה המתאים ב-`src/schema/` בלבד.
